Creamos el servidor con el programa Wamp Server, su instalación y configuración es muy sencilla.

Lanzamos la base de datos utilizando phpmyadmin, que viene incorporado con el programa Wamp Server.
  - Creamos una base de datos con el nombre: api_parking
  - Para inicializar la base creamos el archivos DB.sql con las tablas y columnas principales, así como algunos elementos para rellenarla.

Descargamos la extensión de chrome Insomnia para debugear la api. Nos permite realizar request con Body o Query, ver los headers y tiempos de respuesta, además de exportarlas a curl.

Utilizamos curl de php para hacer test. Creamos un pequeño script para hacer llamadas GET y POST.
